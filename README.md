This is a basic Solana program using the Anchor framework. It logs 'Hello, Solana!' to the chain.

## Build
```bash
anchor build
```

## Test
```bash
anchor test
```

## Project Structure
- `programs/` – Rust smart contracts
- `tests/` – JS/TS test scripts
- `Anchor.toml` – project configuration
